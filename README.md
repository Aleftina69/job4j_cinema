**Cinema Project**


Проект "Cinema" — это веб-приложение для управления информацией о фильмах, реализованное с использованием современных технологий. Приложение позволяет просматривать, добавлять и редактировать данные о фильмах, жанрах и связанных файлах.

**Технологии**

* Spring Boot — фреймворк для создания RESTful и MVC приложений на Java.
* Thymeleaf — серверный шаблонизатор для динамического формирования HTML страниц.
* Bootstrap — CSS-фреймворк для адаптивного и современного дизайна.
* Liquibase — инструмент для управления версиями базы данных и миграций.
* Sql2o — простой и легковесный ORM для работы с базой данных.
* PostgreSQL — реляционная СУБД для хранения данных.

**Описание проекта**

Приложение предоставляет интерфейс для работы с каталогом фильмов. Пользователи могут просматривать список фильмов с подробной информацией, добавлять новые фильмы, редактировать существующие и управлять жанрами.

**Требования**
* Java 17+
* Maven 3.6+
* PostgreSQL 12+
